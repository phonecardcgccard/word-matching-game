# 单词匹配游戏

一个简单有趣的中英文单词匹配游戏，支持拖拽和点击两种操作模式。

## 功能特点

- 支持拖拽和点击两种操作模式
- 实时分数统计
- 错题本功能
- 支持导入自定义单词表
- 支持导出错题
- 音效反馈
- 移动端适配

## 如何使用

1. 选择游戏模式（拖拽/点击）
2. 匹配对应的中英文单词
3. 成功匹配会得到分数奖励
4. 错误匹配会被记录到错题本
5. 可以导出错题进行复习

## 本地开发

1. 克隆仓库
2. 打开 index.html
3. 开始开发
